ニュース・SNS

清心会の最新情報は、SNS・ブログ・掲示板で随時発信しています。行事予定や稽古の様子、休講のお知らせなどは下記の各メディアからご覧ください。

Facebook グループ 清心会空手道場の公式Facebookグループ。日々の活動の様子をお届けします。 → X (Twitter) 清心会公式アカウント。最新のお知らせを発信しています。 → TikTok 演武や稽古風景、大会の様子などを動画で紹介しています。 → YouTube チャンネル 演武・稽古・メディア出演などの動画を公開しています。 → 公式ブログ 稽古や行事の様子を綴る清心会道場の公式ブログです。 → 行事ブログ 大会や審査会、合宿などの行事の様子を掲載しています。 → ニュースブログ 清心会からのお知らせや最新情報を掲載しています。 → 大久保道場ブログ 大久保道場の活動や稽古の様子などを紹介しています。 → 掲示板・連絡事項 休講情報や連絡事項を掲載しています。悪天候時などはこちらをご確認ください。 → 清心会空手行事予定 大会・審査会・合宿など、年間の行事予定を掲載しています。 →

※各リンクは外部サイトが開きます。
