SEISHINKAI WEBSITE REDESIGN — RATIONALE & NEXT STEPS
(working notes for client email)

IMPROVEMENTS OVER THE OLD SITE
------------------------------
- Fully mobile responsive: the site now adapts to phones and tablets
  (hamburger menu, flexible layouts), where the old site was desktop-only.
- Bilingual: complete Japanese and English versions of every page, with a
  one-tap language switch in the header. Search engines are told about both
  versions (hreflang), so Japanese and international visitors each find the
  right one.
- All previous content preserved: every dojo, schedule link, instructor,
  and history entry from the old site has been carried over — nothing lost.
- Clearer, modern layout: the same information is reorganized into cards,
  accordions, and clean sections instead of long unbroken pages.
- Reader-friendly text: the largest blocks of text from the original have
  been summarized and broken up for easy scanning, while keeping most of
  the original wording and all key facts intact.
- Search engine optimization built in:
  - Unique page titles and descriptions on every page, in both languages
    (the old site showed the same title everywhere).
  - Sitemap and robots.txt generated automatically, ready to submit to
    Google Search Console.
  - Structured data (schema.org) so Google understands this is a karate
    organization in Sapporo/Hokkaido.
  - Every image now has descriptive alt text (JA + EN) — better Google
    image search ranking and accessibility for screen readers.
- Fast and low-maintenance: built as a static site (Astro), so pages load
  almost instantly.
- Modern conveniences: smooth scrolling, back-to-top button, image
  slider on the home page, clickable phone-friendly schedule cards.

POSSIBLE FUTURE IMPROVEMENTS
----------------------------
- Updated photography: several photos are older or low resolution; a
  fresh set of high-quality photos (training, dojo exteriors, portraits)
  would lift the whole site.
- Gallery page: a dedicated photo/video gallery of tournaments, gradings,
  and events — easy to add as a new page.
- Google Search Console + Analytics: submit the sitemap and add a GA4
  tag so we can measure visitors and search performance (tag placement is
  already prepared in the template).
- Instructor name romanization: a few instructors' given names have
  ambiguous readings; confirming official romaji with the client would
  let us show full names on the English pages.
- Multilingual expansion: the locale system supports adding further
  languages later if the federation's overseas branches need them.
